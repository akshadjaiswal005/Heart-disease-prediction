# Tisha Models - Machine Learning TAE-1

## Models Included

1. Decision Tree Model  
   - Implementation of Decision Tree Classifier  
   - Includes data preprocessing, feature encoding, and visualization  
   - Performance evaluated using accuracy, confusion matrix, and classification report  

2. Support Vector Machine (SVM) Model  
   - Implementation of SVM (SVC) for classification  
   - Includes kernel comparison (linear, polynomial, sigmoid)  
   - Hyperparameter tuning and performance evaluation  

3. Naive Bayes Model  
   - Implementation using Gaussian Naive Bayes  
   - Efficient probabilistic classification approach  
   - Includes performance analysis and visualization  

## Project Details

- Dataset: UCI Heart Disease Dataset  
- Task: Classification (Heart Disease Prediction)  
- Tools Used: Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn  
- Platform: Google Colab  
- Version Control: GitHub (Collaborative Development)  

## Features of Implementation

- Data preprocessing and cleaning  
- Handling missing values  
- Feature encoding and transformation  
- Model training and testing  
- Performance evaluation using standard metrics  
- Visualization for better understanding
